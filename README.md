# CometPath AI: Graph-Augmented Course Recommendation System
CometPath AI is an intelligent academic advisor platform designed for students at The University of Texas at Dallas (UTD). Unlike standard chatbots, CometPath uses a Hybrid Graph-Vector Architecture to provide mathematically accurate degree audits and career-aligned course recommendations.

By modeling the UTD Academic Catalog as a Knowledge Graph (Neo4j) and utilizing Vector Embeddings (OpenAI), the system can handle complex prerequisite logic while matching a student’s unique professional background (Resume) to the curriculum.
